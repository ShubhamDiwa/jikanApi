# Jikan Anime App

An Android app built using **Jetpack Compose** and **MVVM** that shows popular anime using the Jikan API.

## Features
- Anime list with pagination
- Anime detail screen
- Offline support using Room
- Offline-first architecture
- Network error handling

## Tech Stack
- Kotlin
- Jetpack Compose
- MVVM
- Retrofit
- Room
- StateFlow

## API
- https://api.jikan.moe/v4/top/anime
- https://api.jikan.moe/v4/anime/{id}
